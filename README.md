# Portfólio bilíngue de Business Intelligence — Paulo Victor

Versão estática em português e inglês, pronta para o GitHub Pages.

## Publicação

1. Extraia todo o conteúdo do ZIP.
2. Envie todos os arquivos e a pasta `logos` para a raiz do repositório.
3. No GitHub, acesse **Settings > Pages**.
4. Selecione **Deploy from a branch**, escolha **main** e **/(root)** e salve.
